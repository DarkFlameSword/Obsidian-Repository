**理解：**
Information Gain（信息增益）是**决策树算法**和**特征选择**中的核心概念，用于衡量一个特征对分类任务的**重要性**

$$\begin{aligned}
&IG(S, A) = H(S) - Σ \left(\frac{|S_v|}{|S|} × H(S_v)\right)\\
&v∈Values(A)
\end{aligned}$$

其中：
- $IG(S, A)$: 特征 A 对数据集 S 的信息增益
- $H(S)$: 划分前的熵
- $S_v$: 特征 A 取值为 v 的子集
- $\frac{|S_v|}{|S|}$: 子集权重
- $H(S_v)$: 子集的熵

**举例：**

**数据集**

|天气|温度|湿度|风力|是否打球|
|---|---|---|---|---|
|晴|热|高|弱|❌ No|
|晴|热|高|强|❌ No|
|阴|热|高|弱|✅ Yes|
|雨|温和|高|弱|✅ Yes|
|雨|凉爽|正常|弱|✅ Yes|
|雨|凉爽|正常|强|❌ No|
|阴|凉爽|正常|强|✅ Yes|
|晴|温和|高|弱|❌ No|
|晴|凉爽|正常|弱|✅ Yes|
|雨|温和|正常|弱|✅ Yes|
|晴|温和|正常|强|✅ Yes|
|阴|温和|高|强|✅ Yes|
|阴|热|正常|弱|✅ Yes|
|雨|温和|高|强|❌ No|

**计算初始熵**
```
# 数据集的类别分布
总样本数 = 14
Yes = 9
No = 5

p(Yes) = 9/14 = 0.643
p(No) = 5/14 = 0.357

H(S) = -0.643 × log₂(0.643) - 0.357 × log₂(0.357)
     = -0.643 × (-0.638) - 0.357 × (-1.486)
     = 0.410 + 0.531
     = 0.941
```

**计算特征"天气"的信息增益**
```
天气 = 晴:
    样本: [No, No, No, Yes, Yes]
    统计: 2 Yes, 3 No
    p(Yes) = 2/5 = 0.4, p(No) = 3/5 = 0.6
    H(晴) = -0.4 × log₂(0.4) - 0.6 × log₂(0.6)
          = -0.4 × (-1.322) - 0.6 × (-0.737)
          = 0.529 + 0.442
          = 0.971

天气 = 阴:
    样本: [Yes, Yes, Yes, Yes]
    统计: 4 Yes, 0 No
    H(阴) = 0  # 完全纯净

天气 = 雨:
    样本: [Yes, Yes, No, Yes, No]
    统计: 3 Yes, 2 No
    p(Yes) = 3/5 = 0.6, p(No) = 2/5 = 0.4
    H(雨) = -0.6 × log₂(0.6) - 0.4 × log₂(0.4)
          = 0.971
```

**计算加权平均熵**
```
划分后的熵 = (5/14) × H(晴) + (4/14) × H(阴) + (5/14) × H(雨)
          = (5/14) × 0.971 + (4/14) × 0 + (5/14) × 0.971
          = 0.347 + 0 + 0.347
          = 0.694
```


**计算信息增益**
```
IG(天气) = H(S) - 划分后的熵
        = 0.941 - 0.694
        = 0.247
```

**计算其他特征的信息增益**
```
IG(温度) = 0.029
IG(湿度) = 0.152
IG(风力) = 0.048
IG(天气) = 0.247  ← 最大！
# 结论：选择"天气"作为根节点，因为它的信息增益最大
```
