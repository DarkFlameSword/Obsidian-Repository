# Naïve Bayes
**理解：**
只有一个“因”，有多个“果”
![[Pasted image 20251007220209.png]]
**理解：**
朴素贝叶斯基于两个核心假设：
1. **贝叶斯定理**：利用先验概率和条件概率进行推理
2. **特征独立性假设**（"朴素"的来源）：假设特征之间相互独立

**公式:**
$$P(y, x_1, x_2, \dots , x_n) = P(y)P(x_1|y)P(x_2|y) \dots P(x_n|y)$$

举例:

|天气|温度|风力|外出玩耍|
|---|---|---|---|
|晴朗|炎热|弱|是|
|晴朗|炎热|强|是|
|阴天|炎热|弱|是|
|雨天|温和|弱|是|
|雨天|寒冷|弱|否|
|雨天|寒冷|强|否|
|阴天|寒冷|强|是|
|晴朗|温和|弱|是|
|晴朗|寒冷|弱|是|
|雨天|温和|弱|是|
|晴朗|温和|强|是|
|阴天|温和|强|是|
|阴天|炎热|弱|是|
|雨天|温和|强|否|

现在，如果天气是“晴朗”，温度是“寒冷”，风力是“强”，我们应该预测是否“外出玩耍”？


**计算先验概率**:
- $P(外出玩耍=是)=\frac{9}{14}$
- $P(外出玩耍=否)=\frac{5}{14}$
        
**计算各类别的条件概率**:
- **对于“是”**:
    - P(天气=晴朗∣外出玩耍=是)=4/9
    - P(温度=寒冷∣外出玩耍=是)=2/9
    - P(风力=强∣外出玩耍=是)=4/9
            
- **对于“否”**:
        
    - P(天气=晴朗∣外出玩耍=否)=1/5
            
    - P(温度=寒冷∣外出玩耍=否)=2/5
            
    - P(风力=强∣外出玩耍=否)=3/5
            
**计算后验概率**:
    
- $P(是∣晴朗, 寒冷, 强)∝P(是)⋅P(晴朗∣是)⋅P(寒冷∣是)⋅P(强∣是)=\frac{9}{14}​⋅\frac{4}{9}​⋅\frac{2}{9}​⋅\frac{4}{9}​≈0.028$
        
- $P(否∣晴朗, 寒冷, 强)∝P(否)⋅P(晴朗∣否)⋅P(寒冷∣否)⋅P(强∣否)=\frac{5}{14}​⋅\frac{1}{5}​⋅\frac{2}{5}​⋅\frac{3}{5}​≈0.017$
        
**比较并决策**: 
因为 0.028>0.017，所以朴素贝叶斯分类器会预测“外出玩耍=是”。

# Naïve Bayes Classifier


**举例：**
```
目标：给定特征 X，预测类别 Y

核心思想：
P(Y|X) = P(X|Y) × P(Y) / P(X)

选择概率最大的类别：
Y* = argmax P(Y|X)
      Y
```

**数学公式：**
Assumes conditional independence of the attribute values for different classes
$$\begin{aligned}
c^* = &argmax_c P(c|v_{i1},v_{i2},\dots,v_{in})\\
&argmax_c \Pi_{k=1}^n P(v_{ik}|c)P(c)
\end{aligned}$$
# Laplace smoothing
在朴素贝叶斯中，计算某个特征（如词）在某类别下出现的概率时，如果某个词在训练集某类别中从未出现，那么概率为0。这样一来，在做预测时如果遇到这个词，就会导致整个概率连乘为0，影响分类结果。


**原始概率估计：**
$$
P(w|C)=\frac{N_{w,C}}{N_C}
$$

**加一平滑后：**
$$
PLaplace(w|C)=\frac{N_{w,C}+1}{N_C+V}
$$
- 某类别 C
- 某词 w
- Nw,C：类别 C 中出现词 w 的次数
- NC：类别 C 中所有词的总次数
- V：词表的大小（所有可能的词数）
