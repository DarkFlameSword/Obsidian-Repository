# 系统分析

### Macroscopic Approaches
假设你要研究一罐传染病气体。

- **微观视角（Microscopic）：** 你需要写出每一个气体分子的位置、速度、质量，还要计算它们两两之间碰撞的受力（这需要解海量的牛顿方程，计算量大到宇宙爆炸）。在传染病中，这叫**基于个体的模型（Agent-Based Model, ABM）**
- **宏观视角（Macroscopic）：** 我们根本不在乎某个具体分子在干什么。我们直接测量整个系统的**宏观状态变量（State Variables）**，比如气体的温度、压强、体积，或者在传染病中，我们只看 **当前有多少人感染（$I$）**
### Deterministic Development of the Average
- **真实的微观世界是随机的（Stochastic）：** 一个感染者遇到一个健康人，由于风向或者免疫力的差异，今天可能传染上了，也可能没传染上（这像掷骰子）。如果系统里只有 2 个感染者，这 2 个人如果今天恰好都没出门，疫情可能就直接结束了。这种随机波动（Fluctuations）在系统很小的时候非常致命
- **大数定律与确定性演化：** 但是，当人口基数 $N$ 非常大（比如一百万人）时，根据概率论中的大数定律，那些微观的随机波动会互相抵消。个体的“偶然”变成了总体的“必然”

所以，**均值的确定性演化**指的是：在大样本、平均场的假设下，我们不再去掷骰子算概率，而是直接利用微分方程（ODE），追踪系统**期望值（平均值）**的一条极其平滑、确定的随时间变化的轨迹

只要你给定了初始条件和参数（如 $\beta$, $\gamma$），不管你跑多少次模型，画出来的那条疫情爆发曲线永远是同一条

# SIR Model
- **S (Susceptible，易感人群)：** 还没有生病，体内没有抗体，一旦接触到病毒就有概率被感染的人。在疫情初期，几乎所有人都在这个房间里
- **I (Infectious/Infected，感染人群)：** 已经生病，且携带有活性病毒，能够将病毒传染给易感者（S）的人
- **R (Recovered/Removed，康复/移出人群)：** 已经病愈并获得了永久免疫力，或者不幸因病死亡的人。对于病毒的传播链来说，这类人群已经彻底“退出”了游戏，既不会传染别人，也不会再次被感染

![[Pasted image 20260312231301.png]]

$$\frac{dS}{dt} = -\frac{\beta S I}{N}$$

$$\frac{dI}{dt} = \frac{\beta S I}{N} - \gamma I$$

$$\frac{dR}{dt} = \gamma I$$

- $N$：总人口数（恒定不变，$N = S + I + R$）
- $\beta$ (Beta)：**感染率（Transmission Rate）**。它代表着一个感染者每天能有效接触并成功传染多少个易感者。戴口罩、保持社交距离，本质上就是为了降低 $\beta$
- $\gamma$ (Gamma)：**恢复/移出率（Recovery Rate）**。它代表感染者每天有多大的概率康复。它通常是平均感染周期的倒数（例如，如果平均感染期为 10 天，那么 $\gamma = 0.1$）。研发特效药本质上就是为了提高 $\gamma$

通过这几个参数，我们可以推导出一个在新闻中极其常见的流行病学指标——**基本传染数 $R_0$ (Basic Reproduction Number)**：

$$R_0 = \frac{\beta}{\gamma}$$

它的物理意义非常直观：在一个完全没有免疫力的社会中，平均一个感染者在整个得病期间，会传染给多少个健康人
- 如果 $R_0 > 1$：一个人能传染给多于一个人，疫情就会呈指数级爆发
- 如果 $R_0 < 1$：病毒的传播链会逐渐断裂，疫情最终会自然消亡

# Deneubourg Standard Model

### Double Bridge Experiment
想象一个蚁巢和一个食物源，中间被一个水池隔开。研究人员在蚁巢和食物之间架设了两座长度完全相同的桥（分支 A 和分支 B）

- **微观视角：** 一只蚂蚁走到岔路口，它看不到哪条路更好，只能靠闻地上的信息素浓度来做决定。选定一条路后，它一边走一边在地上留下自己的信息素
- **宏观问题：** 既然两座桥一模一样，最终蚂蚁们是会各占 50% 均匀分布在两条桥上，还是会集中在其中一条桥上？

Deneubourg 没有去追踪每一只蚂蚁的大脑是如何思考的，他使用了我们刚学过的**宏观确定性方法**，建立了一个极其优雅的概率模型

他提出，蚂蚁选择分支 A 的概率 $P_A$，不仅取决于 A 桥上的信息素浓度，还呈现出一种**非线性（Non-linear）** 的关系。其标准数学公式通常长这样：

$$P_A = \frac{(k + A)^n}{(k + A)^n + (k + B)^n}$$

- $A$ 和 $B$：分别是分支 A 和分支 B 上当前积累的信息素量
- $k$：**固有吸引力（或信息素检测阈值）**。即使地上毫无信息素（$A=0, B=0$），蚂蚁也会因为这座桥本身的存在而有一定概率走上去
- $n$：**非线性放大系数**。它代表了蚂蚁对信息素浓度的“敏感度”或跟风程度。如果 $n > 1$，这就构成了一个强大的**正反馈循环（Positive Feedback Loop）**

其宏观的微分方程长这样：
$$\frac{\partial C_i}{\partial t} = \frac{1}{l_i} \cdot \frac{f(C_i)}{f(C_1) + f(C_2)} - \rho C_i$$
**左边：时间的车轮（变化率）**

- **$\frac{\partial C_i}{\partial t}$**：代表第 $i$ 条路径上，信息素浓度 $C_i$ 随时间 $t$ 的**变化率**。
    - 如果结果为正，说明蚂蚁留下信息素的速度大于挥发的速度，这条路越来越“热”
    - 如果结果为负，说明信息素在消散，这条路正在被“遗忘”


**右边第一部分：积累与几何优势（正反馈）**

这部分定义了信息素是如何**增加**的，它由两个极其巧妙的因子相乘得到：

- **$\frac{f(C_i)}{f(C_1) + f(C_2)}$（选择该路径的种群比例）：**
    这正是我们上一节讲的 Deneubourg 非线性概率公式。$f(C_i)$ 通常是信息素浓度的某种 $n$ 次方（例如 $C_i^n$）。它代表了蚂蚁的“盲从效应”：哪条路现有的信息素越浓，下一只蚂蚁选择这条路的概率就呈指数级飙升

- **$\frac{1}{l_i}$（沉积调制量 / 路径长度的倒数）：**
    这是破局的关键！$l_i$ 是这条路径的物理长度。为什么是倒数？

    想象一下，短桥的长度 $l_i$ 很小，长桥的长度 $l_i$ 很大。因为短桥距离近，走短桥的蚂蚁完成一次往返（比如从巢穴到食物再回来）花费的时间更少。在**单位时间**内，短桥上的蚂蚁能跑更多的趟数，也就是能“高频”地涂抹信息素。所以，路径越短，信息素的初始沉积率 $\frac{1}{l_i}$ 就越高


**右边第二部分：大自然的橡皮擦（负反馈）**

这部分定义了信息素是如何**减少**的：
- **$-\rho C_i$（蒸发项）：**
    自然界中，气味是会挥发的。$\rho$ (Rho) 是蒸发系数。当前浓度 $C_i$ 越高，挥发掉的绝对量就越大
    
    **蒸发的意义极其重大：** 它是系统的“刷新键”。如果没有蒸发，即使后面食物源变了，蚂蚁也会死死守着旧路。蒸发机制清除了过时的信息，让系统能够随着环境动态适应

### Symmetry Breaking

这个模型之所以被称为“标准模型”，是因为它完美地用数学语言解释了复杂系统中的一个核心现象：**对称性破缺**

结合我们刚才讨论的宏观与微观概念：

1. **初始的随机性（微观摇摆）：** 刚开始，两座桥上都没有信息素（$A=B=0$），蚂蚁选择 A 或 B 的概率都是 50%。前几只蚂蚁的走向完全是随机的（比如恰好有 3 只走了 A，2 只走了 B）
2. **正反馈的放大：** 因为 A 桥上多了一只蚂蚁，留下的信息素稍微多了一点点。下一只蚂蚁走到路口时，根据上面的公式，$P_A$ 变得略大于 50%
3. **宏观的确定性演化：** 随着蚂蚁数量（$N$）不断增加，这个微小的随机偏差被非线性的 $n$ 次方疯狂放大。“走 A 的人越多 $\rightarrow$ A 的信息素越浓 $\rightarrow$ 选 A 的概率越大”
4. **最终结果（分岔 Bifurcation）：** 最终，整个蚁群会“锁定”在其中一座桥上（比如 100% 走 A 桥），而另一座桥被完全遗弃

### 核心意义

一个群体想要展现出高度一致、仿佛经过深思熟虑的“智能决策”（比如所有蚂蚁都找到并锁定了同一条路线），根本不需要一个“蚁后”来发号施令

只需要满足三个条件：
1. **个体遵循极其简单的局部规则**（闻到哪边味道重就偏向哪边）
2. **个体之间有间接的交流机制**（信息素）
3. **存在非线性的正反馈**（公式中的 $n$ 次方）

这种将微观层面的局部随机交互，转化为宏观层面具有高度确定性、结构性的集体行为的模型，后来深刻地启发了计算机科学领域的**蚁群算法（Ant Colony Optimization, ACO）**，被广泛应用于网络路由寻址、物流路径规划等极其复杂的工程问题中
