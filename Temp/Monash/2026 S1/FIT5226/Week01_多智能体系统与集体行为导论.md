# 多智能体系统与集体行为导论

> **Course**: FIT5226 | **Week**: 1 | **Source**: Wk 1 Introduction and Overview.pdf

## Key Concepts

1. **多智能体系统**：由多个相互作用的自主智能体组成的系统，表现出集体行为
2. **集体行为**：简单个体遵循简单规则，在群体层面涌现出复杂模式
3. **最大流最小割**：组合优化中的基本问题，涉及在容量有限的运输网络中确定从源节点到汇节点的最大可能流量
4. **数据密集型计算生态学**：结合AI数据提取、实验室实验、数学行为建模和高性能模拟的研究领域
5. **自然多智能体系统示例**：从社会性昆虫到经济市场等广泛存在的集体行为现象

### 最大流与最小割问题
- **最大流问题**：在容量有限的运输网络中，确定从源节点a到汇节点b的最大可能流量
- **最小割问题**：与最大流对偶的问题
- **重要性**：组合优化中的基本问题
- **应用领域**：
  - 调度（航空、铁路等）
  - 运输问题
  - 匹配问题
  - 网络路由
  - 其他许多问题可以归约为最大流问题

### 数据密集型计算生态学
**研究框架**：
1. **AI数据提取**：使用人工智能技术从观测数据中提取信息
2. **实验室实验**：受控环境下的行为实验
3. **数学行为建模**：建立描述个体和群体行为的数学模型
4. **高性能模拟**：使用计算模拟研究大规模系统行为

### 什么是多智能体系统？
- **智能体**：具有自主性、感知环境、做出决策并采取行动的实体
- **多智能体系统**：由多个智能体组成的系统，智能体之间通过相互作用产生协调或竞争行为
- **核心特征**：
  - 分布式控制：没有中央协调者
  - 局部交互：智能体只与邻近个体交互
  - 简单规则：个体行为规则简单
  - 涌现性：群体层面出现复杂模式

### 集体行为基本原理
1. **自组织**：系统从无序中自发产生有序结构
2. **涌现性**：整体大于部分之和，群体行为不能从个体行为简单推导
3. **正反馈**：加强特定行为或模式
4. **负反馈**：抑制过度增长或偏离
5. **随机性**：个体行为中的随机成分促进探索和适应

### 研究方法
1. **数学建模**：微分方程、随机过程、博弈论
2. **计算模拟**：基于代理的建模、蒙特卡洛方法
3. **数据分析**：从观测数据中提取模式和规律
4. **理论分析**：稳定性分析、相变理论、均值场理论

### 应用领域
- **交通系统**：车辆流量优化、拥堵管理
- **机器人集群**：协作完成任务
- **社交网络**：信息传播、意见动态
- **金融市场**：交易者行为、价格形成
- **生态系统**：种群动态、物种相互作用
- **城市系统**：人流、资源分配

## Key Terms

| Term | Definition |
|------|-----------|
| 多智能体系统 | 由多个相互作用的自主智能体组成的系统，智能体具有感知、决策和行动能力 |
| 集体行为 | 个体遵循简单规则，在群体层面自发形成的协调模式 |
| 涌现性 | 复杂模式或行为从简单个体的相互作用中产生，不能从个体行为简单推导 |
| 自组织 | 系统在没有外部指令或中央控制的情况下，从无序中自发形成有序结构 |
| 最大流问题 | 在容量有限的网络中寻找从源点到汇点的最大流量的问题 |
| 最小割问题 | 寻找将网络分为两个部分的最小容量切割，与最大流问题对偶 |
| 基于代理的建模 | 通过模拟个体（代理）的行为和相互作用来研究系统整体行为的方法 |
