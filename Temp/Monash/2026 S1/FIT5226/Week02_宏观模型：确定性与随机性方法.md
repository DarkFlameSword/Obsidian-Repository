# 宏观模型：确定性与随机性方法

> **Course**: FIT5226 | **Week**: 2 | **Source**: 宏观确定性模型、宏观随机性模型、实验课材料

## Key Concepts

1. **宏观方法**：将群体视为整体，建模宏观属性而非个体行为
2. **确定性模型**：使用微分方程等确定性方法描述系统动态
3. **随机性模型**：考虑随机因素，使用概率描述系统状态变化
4. **马尔可夫链**：离散时间、离散状态的随机过程，具有无记忆性
5. **传染过程**：描述疾病、信息或行为在群体中传播的模型
6. **谢林模型**：经典的隔离模型，展示个体偏好如何导致宏观隔离模式

### 宏观方法
**基本思想**：
- 将所有智能体视为平等且可交换的
- 将群体划分为不同类别的子群体
- 建模类别之间的转移率

**优势**：
- 计算效率高
- 适合大规模系统
- 提供群体层面的洞察

**局限性**：
- 忽略个体差异
- 无法捕捉微观互动细节

### 确定性宏观模型
**连续时间模型**：
- 常微分方程（ODEs）描述状态变量随时间的变化
- 示例：SIR传染病模型
  - S：易感者
  - I：感染者  
  - R：康复者
  - 方程：dS/dt = -βSI, dI/dt = βSI - γI, dR/dt = γI

**离散时间模型**：
- 差分方程描述离散时间步长的状态变化
- 示例：逻辑斯蒂映射

### 随机性宏观模型
**随机过程定义**：索引随机变量族，描述系统在不同时间处于不同状态的概率变化

**为什么需要随机模型？**
1. **固有随机性**：系统本身具有随机成分
2. **测量不确定性**：观测中的噪声
3. **模型不确定性**：简化假设带来的不确定性
4. **小群体效应**：群体规模小时，随机波动显著

### 马尔可夫链
**定义**：具有马尔可夫性质的离散时间随机过程
- **状态空间**：系统可能处于的状态集合
- **转移概率**：从状态i转移到状态j的概率P(i→j)
- **无记忆性**：未来状态只依赖于当前状态，不依赖于过去历史

**性质**：
- 齐次性：转移概率不随时间变化
- 遍历性：从任何状态出发，最终都能到达任何其他状态
- 稳态分布：长期运行后状态的概率分布

### 传染过程
**基本SI模型**：
- 两种状态：易感者(S)和感染者(I)
- 感染率：βSI
- 恢复率：γI（在SIR模型中）

**关键概念**：
- 基本再生数R₀：一个感染者在完全易感群体中引起的平均新感染数
- 群体免疫阈值：需要免疫以阻止传播的群体比例（1 - 1/R₀）
- 相变：传播阈值处的行为突变

### 谢林模型（实验课）
**背景**：Thomas Schelling提出的隔离模型，展示个体温和的偏好如何导致严重的宏观隔离

**模型规则**：
1. 智能体居住在网格世界中
2. 每个智能体有类型（如红/蓝）
3. 每个智能体检查周围邻居
4. 如果不满意（同类邻居比例低于阈值），则移动到随机空闲位置
5. 重复直到稳定

**涌现现象**：
- 即使个体只要求30-40%的同类邻居，最终也会形成高度隔离的模式
- 个体偏好与宏观结果之间的非线性关系
- 临界阈值的存在

### 离散vs连续建模
| 方面 | 离散模型 | 连续模型 |
|------|----------|----------|
| 时间 | 离散时间步长 | 连续时间 |
| 状态 | 离散状态空间 | 连续状态空间 |
| 工具 | 马尔可夫链、差分方程 | 微分方程、随机微分方程 |
| 优点 | 计算简单、直观 | 解析处理方便、平滑 |
| 应用 | 小群体、离散事件 | 大群体、连续动态 |

### 噪声的重要性
1. **探索**：帮助系统探索状态空间
2. **相变**：在临界点附近放大涨落
3. **多样性**：维持群体多样性，防止过早收敛
4. **稳健性**：使系统对扰动更具弹性

```python
class SchellingModel:
    def __init__(self, grid_size, n_agents, threshold):
        self.grid_size = grid_size
        self.grid = np.zeros((grid_size, grid_size))
        self.threshold = threshold
        # 初始化智能体位置和类型
        
    def step(self):
        # 实现模型的一步更新
        for agent in self.agents:
            if not self.is_satisfied(agent):
                self.move_agent(agent)
                
    def is_satisfied(self, agent):
        # 检查智能体是否满意当前位置
        neighbors = self.get_neighbors(agent)
        similar = sum(1 for n in neighbors if n.type == agent.type)
        return similar / len(neighbors) >= self.threshold
```

## Summary
第二周介绍了多智能体系统研究中的宏观建模方法，包括确定性和随机性两种范式。确定性方法使用微分方程描述群体层面的平均行为，而随机性方法通过马尔可夫链等工具考虑噪声和随机波动。传染过程作为经典案例，展示了如何用简单模型捕捉复杂传播动态。实验课通过谢林模型揭示了微观个体偏好与宏观群体模式之间的非线性关系，为后续基于代理的建模和强化学习打下基础。理解这些宏观方法对于分析大规模多智能体系统的集体行为至关重要。

## Key Terms

| Term | Definition |
|------|-----------|
| 宏观模型 | 将群体视为整体，建模类别之间转移率的建模方法 |
| 确定性模型 | 使用精确数学方程描述系统行为，不含随机因素的模型 |
| 随机性模型 | 考虑随机因素，使用概率描述系统状态变化的模型 |
| 马尔可夫链 | 具有无记忆性的离散时间随机过程，未来状态只依赖于当前状态 |
| 传染过程 | 描述疾病、信息或行为在群体中传播的数学模型 |
| 谢林模型 | 展示个体居住偏好如何导致群体隔离的经典多智能体模型 |
| 基本再生数 | 一个感染者在完全易感群体中引起的平均新感染数 |
| 稳态分布 | 马尔可夫链长期运行后状态的概率分布 |
