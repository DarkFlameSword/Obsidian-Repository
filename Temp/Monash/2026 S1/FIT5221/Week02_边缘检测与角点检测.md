# 边缘检测与角点检测

> **Course**: FIT5221 | **Week**: 2 | **Source**: Lecture-2.pdf

## Key Concepts

1. **边缘检测**：识别图像中强度的突然变化（不连续性），提取语义和形状信息
2. **导数与边缘**：边缘对应于图像强度函数导数的极值点
3. **梯度向量**：指向强度变化最迅速方向的向量，包含幅值和方向信息
4. **卷积实现导数**：使用滤波器核近似计算图像偏导数
5. **角点检测**：识别图像中在两个方向上都有显著强度变化的点（兴趣点）

## Detailed Notes

### 边缘检测概述
- **目标**：识别图像中的突然变化（不连续性）
- **重要性**：
  - 大多数语义和形状信息可以通过边缘编码
  - 比像素表示更紧凑
  - 为后续特征提取和对象识别提供基础

### 导数与边缘的关系
- **边缘定义**：图像强度函数快速变化的位置
- **数学表示**：边缘对应于强度函数一阶导数的极值点
- **强度函数**：沿水平扫描线的图像强度剖面
- **导数极值**：边缘出现在导数最大值或最小值处

### 使用卷积计算导数
**偏导数定义**：
对于二维函数f(x,y)，x方向的偏导数为：
\[ \frac{\partial f(x,y)}{\partial x} = \lim_{\epsilon \to 0} \frac{f(x+\epsilon, y) - f(x, y)}{\epsilon} \]

**离散近似**：
对于离散数据，使用有限差分近似：
\[ \frac{\partial f(x,y)}{\partial x} \approx \frac{f(x+1, y) - f(x, y)}{1} \]

**滤波器实现**：
- **相关操作**：使用滤波器[-1, 1]
- **卷积操作**：使用滤波器[1, -1]

### 图像偏导数
- **x方向偏导数**：检测垂直边缘
- **y方向偏导数**：检测水平边缘
- **滤波器核**：
  - x方向：[-1, 1]（相关）或 [1, -1]（卷积）
  - y方向：垂直方向的类似滤波器

### 梯度向量
- **定义**：梯度向量指向强度变化最迅速的方向
\[ \nabla f = \begin{bmatrix} \frac{\partial f}{\partial x} \\ \frac{\partial f}{\partial y} \end{bmatrix} \]
- **梯度幅值**：变化的强度
\[ |\nabla f| = \sqrt{\left(\frac{\partial f}{\partial x}\right)^2 + \left(\frac{\partial f}{\partial y}\right)^2} \]
- **梯度方向**：变化的方向
\[ \theta = \arctan\left(\frac{\partial f/\partial y}{\partial f/\partial x}\right) \]

### 边缘检测滤波器
1. **Sobel算子**：
   - x方向：[[-1,0,1],[-2,0,2],[-1,0,1]]
   - y方向：[[-1,-2,-1],[0,0,0],[1,2,1]]
   - 优点：对噪声具有一定的抵抗能力

2. **Prewitt算子**：
   - x方向：[[-1,0,1],[-1,0,1],[-1,0,1]]
   - y方向：[[-1,-1,-1],[0,0,0],[1,1,1]]

3. **Scharr算子**：改进的Sobel算子，具有更好的旋转对称性

### Canny边缘检测器
**经典算法步骤**：
1. **噪声抑制**：使用高斯滤波器平滑图像
2. **梯度计算**：计算图像梯度幅值和方向
3. **非极大值抑制**：沿梯度方向保留局部最大值，细化边缘
4. **双阈值处理**：
   - 高阈值：强边缘
   - 低阈值：弱边缘
   - 连接分析：弱边缘只有在连接到强边缘时才保留

### 角点检测
- **定义**：在两个方向上都有显著强度变化的点
- **也称为**：兴趣点、特征点
- **特点**：
  - 对旋转、尺度、光照变化具有一定不变性
  - 适合作为图像匹配的基准点

### Harris角点检测器
**核心思想**：
- 分析窗口在图像上移动时的强度变化
- 计算自相关矩阵（结构张量）：
\[ M = \sum_{x,y} w(x,y) \begin{bmatrix} I_x^2 & I_xI_y \\ I_xI_y & I_y^2 \end{bmatrix} \]
其中I_x和I_y是x和y方向的图像导数

**角点响应函数**：
\[ R = \det(M) - k \cdot \operatorname{trace}(M)^2 \]
其中k是经验参数（通常0.04-0.06）

**判断准则**：
- R > 0：角点
- R ≈ 0：边缘
- R < 0：平坦区域

### 应用领域
- **图像配准**：将多幅图像对齐
- **对象识别**：基于特征点的匹配
- **三维重建**：从多视角图像重建三维结构
- **运动跟踪**：跟踪视频序列中的特征点
- **全景拼接**：创建全景图像

## Code Examples

```python
import cv2
import numpy as np
import matplotlib.pyplot as plt

# 读取图像
img = cv2.imread('image.jpg', cv2.IMREAD_GRAYSCALE)

# Sobel边缘检测
sobel_x = cv2.Sobel(img, cv2.CV_64F, 1, 0, ksize=3)
sobel_y = cv2.Sobel(img, cv2.CV_64F, 0, 1, ksize=3)
gradient_magnitude = np.sqrt(sobel_x**2 + sobel_y**2)

# Canny边缘检测
edges_canny = cv2.Canny(img, 50, 150)

# Harris角点检测
gray = np.float32(img)
dst = cv2.cornerHarris(gray, blockSize=2, ksize=3, k=0.04)
dst = cv2.dilate(dst, None)
img[dst > 0.01 * dst.max()] = [0, 0, 255]  # 标记角点为红色
```

## Summary
第二周深入探讨了计算机视觉中的两个基础但至关重要的技术：边缘检测和角点检测。边缘检测通过识别图像中的强度不连续性来提取形状信息，为后续分析提供紧凑表示。角点检测则识别具有独特局部特征的点，这些点对图像变换具有较好的不变性，是许多高级视觉任务的基础。从导数计算到梯度分析，再到Harris角点检测器的数学原理，本周内容为理解传统计算机视觉方法奠定了坚实的理论基础。

## Key Terms

| Term | Definition |
|------|-----------|
| 边缘 | 图像中强度函数快速变化的位置，对应于导数极值点 |
| 梯度 | 指向强度变化最迅速方向的向量，包含幅值和方向信息 |
| Sobel算子 | 用于边缘检测的离散微分算子，对噪声具有一定抵抗能力 |
| Canny检测器 | 多阶段边缘检测算法，包括噪声抑制、梯度计算、非极大值抑制和双阈值处理 |
| 角点 | 在两个方向上都有显著强度变化的图像点，也称为兴趣点 |
| Harris检测器 | 基于自相关矩阵的角点检测算法，通过角点响应函数判断 |
| 非极大值抑制 | 边缘细化技术，只保留梯度方向上的局部最大值 |
