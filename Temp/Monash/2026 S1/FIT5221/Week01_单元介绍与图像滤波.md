### 图像与点操作
- **像素级操作**：直接对单个像素值进行操作
- **常见点操作**：
  - 亮度调整
  - 对比度调整
  - 颜色空间转换
  - 阈值处理
- **数学表示**：g(x,y) = T[f(x,y)]，其中T是变换函数

### 图像滤波
- **邻域操作**：基于像素及其周围像素的值进行计算
- **卷积**：使用卷积核（滤波器）在图像上滑动计算
- **常见滤波器类型**：
  1. **平滑滤波器**：模糊图像，减少噪声
    - 均值滤波
    - 高斯滤波
  2. **锐化滤波器**：增强边缘和细节
    - 拉普拉斯滤波器
    - 索贝尔滤波器
  3. **边缘检测滤波器**：识别图像中的边缘
    - Canny边缘检测
    - Prewitt算子

### 卷积操作详解
- **卷积核**：小矩阵（通常3×3、5×5等）
- **操作过程**：
  1. 将卷积核中心对准图像中的每个像素
  2. 将核的每个元素与对应像素值相乘
  3. 将所有乘积相加
  4. 将结果作为输出图像中对应位置的新像素值
- **边界处理**：零填充、复制填充、镜像填充等

### 高斯滤波
- **特点**：使用高斯函数作为权重的平滑滤波器
- **优点**：
  - 在平滑同时更好地保留边缘
  - 权重随距离增加而减小
- **应用**：去噪、尺度空间表示

### 图像处理应用领域
- **医学影像**：肿瘤检测、组织分析
- **自动驾驶**：道路检测、障碍物识别
- **安全监控**：人脸识别、异常检测
- **娱乐媒体**：特效处理、图像增强

## Code Examples
本周讲座主要介绍概念，但可能会在实验课中涉及以下Python/OpenCV代码：

```python
import cv2
import numpy as np

# 读取图像
img = cv2.imread('image.jpg', cv2.IMREAD_GRAYSCALE)

# 均值滤波
kernel = np.ones((3,3), np.float32) / 9
mean_filtered = cv2.filter2D(img, -1, kernel)

# 高斯滤波
gaussian_filtered = cv2.GaussianBlur(img, (5,5), 0)

# 边缘检测（Sobel）
sobel_x = cv2.Sobel(img, cv2.CV_64F, 1, 0, ksize=5)
sobel_y = cv2.Sobel(img, cv2.CV_64F, 0, 1, ksize=5)
```

## Key Terms

| Term | Definition |
|------|-----------|
| 点操作 | 直接对单个像素值进行的图像处理操作 |
| 卷积 | 使用滤波器核在图像上滑动计算的数学操作 |
| 卷积神经网络 | 专门设计用于处理网格状数据（如图像）的深度学习架构 |
| 高斯滤波 | 使用高斯函数作为权重的图像平滑技术 |
| 边缘检测 | 识别图像中强度或颜色显著变化区域的过程 |
| 兴趣点 | 图像中具有独特局部特征的像素位置 |
| 计算机视觉 | 使计算机能够从数字图像或视频中获取高层次理解的学科 |
