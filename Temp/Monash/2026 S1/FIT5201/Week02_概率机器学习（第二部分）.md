# 概率机器学习（第二部分）

> **Course**: FIT5201 | **Week**: 2 | **Source**: 模块1第B部分 - 概率机器学习

## Key Concepts

1. **不确定性**：机器学习中的核心概念，反映对训练数据上学到模型的不确定性
2. **概率论基础**：为衡量和操作不确定性提供数学框架
3. **频率主义方法**：基于重复采样和自助法（bootstrap）的统计推断
4. **贝叶斯方法**：使用概率表示参数不确定性，结合先验知识和数据
5. **贝叶斯机器学习**：将贝叶斯统计应用于机器学习模型

## Detailed Notes

### 模块1：机器学习要素 - 第B部分
**目标**：
- 回顾概率论基础
- 学习概率论如何帮助捕捉机器学习中的关键概念——"不确定性"
  - 频率主义方法（自助法）
  - 贝叶斯方法（概率）
- 学习机器学习的贝叶斯方法

### 机器学习中的不确定性
- **定义**：不确定基于训练数据学到的模型是否是一个好模型
- **数学表示**：P(w|D) - 参数w在给定数据D下的概率分布
- **核心问题**：如果使用不同的数据集D，参数w会如何变化？

### 不确定性的来源
1. **噪声**：数据中的随机变化或测量误差
2. **数据集有限大小**：有限的样本无法完全代表总体

### 为什么学习概率论？
- 为衡量和操作"不确定性"提供框架
- 应用概率论来捕捉学习模型和做出预测时的不确定性
- 涵盖两种主要统计范式：
  - 频率主义统计
  - 贝叶斯统计

### 概率论基础
**核心主题**：
- 离散和连续随机变量
- 概率分布
- 随机过程（不确定过程的数学抽象）

**示例**：抛硬币实验
- 如何用数学模型表示抛硬币事件？
- 随机变量：正面或反面的结果
- 概率分布：伯努利分布

### 频率主义方法
- **核心思想**：概率定义为长期频率的极限
- **参数估计**：点估计（如最大似然估计）
- **不确定性量化**：置信区间、假设检验
- **自助法**：通过重采样从原始数据集中创建多个新数据集，用于估计统计量的分布

### 贝叶斯方法
- **核心思想**：概率表示主观信念程度
- **贝叶斯定理**：P(θ|D) ∝ P(D|θ) × P(θ)
  - 后验分布 ∝ 似然 × 先验分布
- **参数**：被视为随机变量，具有概率分布
- **先验知识**：在观察数据之前对参数的信念
- **后验分布**：结合先验知识和观察数据后的参数分布

### 贝叶斯机器学习
- **模型参数**：具有先验分布
- **学习过程**：通过贝叶斯推理从数据中更新参数分布
- **预测**：考虑所有可能参数值的加权平均（后验预测分布）
- **优势**：
  - 自然地处理不确定性
  - 避免过拟合（通过正则化先验）
  - 结合领域知识（通过先验分布）
  - 提供完整的概率预测

### 频率主义 vs 贝叶斯
| 方面 | 频率主义 | 贝叶斯 |
|------|----------|--------|
| 概率定义 | 长期频率 | 主观信念程度 |
| 参数 | 固定未知常数 | 具有分布的随机变量 |
| 推断 | 基于重复采样 | 基于先验和似然 |
| 不确定性 | 置信区间、p值 | 后验分布、可信区间 |
| 计算 | 通常更简单 | 可能需要近似方法 |

### 实际应用
- **贝叶斯线性回归**：为权重分配先验分布
- **贝叶斯神经网络**：权重具有概率分布
- **高斯过程**：函数的概率分布
- **概率图模型**：使用概率表示变量间关系

## Summary
第二周深入探讨了机器学习中的不确定性概念，并介绍了概率论作为处理不确定性的数学框架。课程对比了频率主义和贝叶斯两种统计范式，重点介绍了贝叶斯方法如何将先验知识与观察数据结合，为参数和预测提供完整的概率分布。贝叶斯机器学习通过自然处理不确定性和避免过拟合，在许多现代应用中显示出优势，为后续学习更复杂的概率模型奠定基础。

## Key Terms

| Term | Definition |
|------|-----------|
| 不确定性 | 对基于有限噪声数据学到的模型的不确定程度 |
| 频率主义 | 将概率定义为长期频率的统计范式 |
| 贝叶斯方法 | 将概率表示信念程度，使用贝叶斯定理更新信念的统计范式 |
| 先验分布 | 在观察数据之前对模型参数的信念分布 |
| 后验分布 | 结合先验知识和观察数据后的参数分布 |
| 似然函数 | 给定参数值下观察数据的概率 |
| 自助法 | 通过从原始数据集中有放回地重采样来估计统计量分布的方法 |
| 贝叶斯定理 | P(θ\|D) = P(D\|θ) × P(θ) / P(D) |
