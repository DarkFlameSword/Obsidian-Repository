### 什么是机器学习？
- **人类学习**：从经验中学习
- **机器学习**：从数据中学习（传感器数据、输入数据等）
- **目的**：自动化 + 从大数据中学习模式

### 机器学习流程
1. **数据收集**：获取原始数据
2. **数据整理**：预处理数据，获取输入变量/属性
3. **模型训练**：学习属性和输出之间的函数关系
4. **模型评估**：测试模型性能
5. **部署应用**：将模型应用于新数据

### 机器学习的关键概念
1. **监督学习**：使用带标签的训练数据（分类、回归）
2. **无监督学习**：从无标签数据中发现模式（聚类、降维）
3. **强化学习**：通过试错和奖励机制学习
4. **特征工程**：从原始数据中提取有用特征
5. **模型选择**：根据问题选择合适算法

### 如何选择好的学习模型？
- **过拟合**：模型过于复杂，在训练数据上表现好但泛化能力差
- **欠拟合**：模型过于简单，无法捕捉数据中的模式
- **偏差-方差权衡**：平衡模型复杂度和泛化能力
- **交叉验证**：评估模型泛化性能的技术
- **正则化**：防止过拟合的技术

## Key Terms

| Term | Definition |
|------|-----------|
| 机器学习 | 从数据中学习模式，无需显式编程的系统 |
| 监督学习 | 使用带标签的训练数据来学习输入到输出的映射 |
| 无监督学习 | 从无标签数据中发现内在结构和模式 |
| 强化学习 | 通过与环境交互和接受奖励/惩罚来学习决策策略 |
| 过拟合 | 模型过于复杂，过度拟合训练数据中的噪声，泛化能力差 |
| 欠拟合 | 模型过于简单，无法捕捉数据中的基本模式 |
| 特征工程 | 从原始数据中提取、转换和选择特征的过程 |
| 交叉验证 | 评估模型泛化性能的统计方法，将数据分为训练集和验证集 |
