# Regularization
| **λ 取值**            | **模型复杂度** | **拟合状态**      | **Bias** | **Variance** |
| ------------------- | --------- | ------------- | -------- | ------------ |
| **过大 $\uparrow$**   | 低（僵化）     | Under-fitting | 高        | 低            |
| **适中**              | 适中        | Good Fit      | 适中       | 适中           |

# Resampling

| **特性**    | **Bootstrap**            | **K-Fold Cross-Validation**            |
| --------- | ------------------------ | -------------------------------------- |
| **抽样方式**  | **有放回**（样本可能重复）          | **无放回**（样本互斥）                          |
| **数据利用率** | 每次使用约 63.2% 的唯一数据        | 每次使用 $\frac{K-1}{K}$ 的数据（如 10 折就是 90%） |
| **适用场景**  | 原始**数据量极小**，或用于构建随机森林    | **数据量适中或充足**，评估模型泛化能力的业界标准             |
| **方差与偏差** | 方差小，但由于没有用到全部样本，可能引入一些偏差 | 偏差小，评估结果更贴近模型在全量数据上的真实表现               |
