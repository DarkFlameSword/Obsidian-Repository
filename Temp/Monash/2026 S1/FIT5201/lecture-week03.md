# Basis Function

$$y(w,x) = w_0+w_1 x + w_2x^2 + w_3x^3\dots w_Mx^M$$
**直觉理解：**
基函数是对输入的一种“特征映射”，把原始输入 $x$ 映射到一个新的特征空间 $\phi(x)$。然后在新空间里做线性模型，就可以表现出对原始输入的非线性关系

**举例：**
假设我们只有一个输入特征 $(x)$，想拟合一条“弯弯的曲线”（比如真实关系接近于 $y = x^3$），但我们的模型本身只提供“线性组合”

最普通的线性回归是：

$f(x)=w_0+w_1x$
- 这只能画成一条直线
- 如果真实关系是弯曲的，这个模型会拟合得很差

**引入多项式基函数：**

$$
\begin{align}
& ϕ_0(x)=1\\
& ϕ_1(x)=x\\
& ϕ_2(x)=x^2\\
& ϕ_3(x)=x^3
\end{align}
$$

然后让模型变成：

$f(x)=w_0ϕ_0(x)+w_1ϕ_1(x)+w_2ϕ_2(x)+w_3ϕ_3(x)$

也就是：

$f(x)=w_0+w_1x+w_2x^2+w_3x^3$

- 对 $\phi_j(x)$（这些基函数）来说，模型是**线性组合**；
- 对原始 $(x)$ 来说，这就是一个**三次多项式**，可以拟合出弯弯的曲线。

**这里的 ($\phi_0, \phi_1, \phi_2, \phi_3$) 就是“多项式基函数”**

---

### Gaussian basis functions
$ϕj(x)=exp⁡(−\frac{(x−μj)^2}{2σ^2})$

- $\mu_j$：第 j 个高斯的“中心”（峰值位置）
- $\sigma$：控制“宽度”（标准差）

你可以把每个 $\phi_j(x)$ 想象成一个“钟形的小山包”，在 $x = \mu_j$ 附近值最大（≈1），远离时迅速衰减到 0。

### 1.1 具体例子

假设我们有 3 个 Gaussian 基函数：

$$
\begin{align}
&ϕ_1(x)=exp⁡(−\frac{(x−0)2}{2})(μ_1=0,σ=1) \\
&ϕ_2(x)=exp⁡(−\frac{(x−2)^2}{2})(μ_2=2,σ=1) \\
&ϕ_3(x)=exp⁡(−\frac{(x−4)^2}{2})(μ_3=4,σ=1)\\
\end{align}
$$

特点：

- 在 $x \approx 0$ 附近，$\phi_1(x) ≈ 1$，其它两个很小
- 在 $x \approx 2$ 附近，$\phi_2(x) ≈ 1$
- 在 $x \approx 4$ 附近，$\phi_3(x) ≈ 1$

如果你用它们做一个模型：

$f(x)=w_1ϕ_1(x)+w_2ϕ_2(x)+w_3ϕ_3(x)$

- 就是在三座“钟形小山”上做线性组合。
- 通过调整 $w_1, w_2, w_3$，可以拼出各种**局部起伏的曲线**，对“局部结构”很敏感。
- 多个 Gaussian 铺在不同位置，就像把 x 轴分成好多“区域”，每个区域有自己的响应。

常见用途：**RBF 网络、核方法、局部加权回归等**

---

### Sigmoidal basis function
最常见的是 **logistic sigmoid**：

ϕ(x)=σ(x)=11+e−x

它的图像是一个平滑的 S 形：

- 当 (x \to -\infty)，(\phi(x) \to 0)
- 当 (x \to +\infty)，(\phi(x) \to 1)
- 在 (x = 0) 时，(\phi(0) = 0.5)

### 2.1 具体例子：带偏移的 sigmoid 基函数

在做基函数时，我们常常对输入做线性变换：

ϕj(x)=σ(ajx+bj)

- (a_j)：控制“斜率”有多陡；
- (b_j)：控制 S 型函数在 x 轴上的平移。

举例：定义两个 sigmoidal 基函数

ϕ1(x)=σ(x−0)=11+e−x(以 0 为中心) ϕ2(x)=σ(x−2)=11+e−(x−2)(以 2 为中心)

直观效果：

- (\phi_1(x))：在 (x=0) 附近急剧从 0 变到 1（大约在 [-4,4] 内变化明显）。
- (\phi_2(x))：整条 S 曲线向右平移 2 个单位，在 (x=2) 附近发生快速变化。

再构造一个模型：

f(x)=w1ϕ1(x)+w2ϕ2(x)

- 在很小的 x（比如 (x \ll 0)）时，两者都接近 0，所以 (f(x) \approx 0)。
- 在很大的 x（比如 (x \gg 2)）时，两者都接近 1，所以 (f(x) \approx w_1 + w_2)。
- 在 0 和 2 附近，函数会分两段发生“S 型上升”，可以做出**阶梯状、分段式的非线性曲线**。

Sigmoid 基函数常用于：

- 早期的前馈神经网络（MLP）的隐藏层激活；
- 做“从 0 到 1 的平滑过渡”（例如概率输出、门控结构）。

---

### Hyperbolic tangent function
**定义：**

tanh⁡(x)=ex−e−xex+e−x

和 sigmoid 很像，也是 S 形，但输出范围不同：

- sigmoid：((0, 1))
- tanh：((-1, 1))

性质：

- 当 (x \to -\infty)，(\tanh(x) \to -1)
- 当 (x \to +\infty)，(\tanh(x) \to 1)
- (\tanh(0) = 0)（关于原点对称）

### .1 具体例子：tanh 作为基函数

和 sigmoid 一样，我们通常带线性变换：

ϕj(x)=tanh⁡(ajx+bj)

举例，定义两个 tanh 基函数：

ϕ1(x)=tanh⁡(x) ϕ2(x)=tanh⁡(2x−3)

- (\phi_1(x))：在 0 附近从 -1 平滑过渡到 1。
- (\phi_2(x))：斜率更陡（因为系数 2），并且在 (x \approx 1.5) 附近是中点（0），在那附近变化最快。

如果构造模型：

f(x)=w1ϕ1(x)+w2ϕ2(x)

- 通过不同中心、不同斜率的 tanh 的线性组合，可以拼出复杂的**波浪状曲线**。
- 因为 tanh 输出有正有负（对称于 0），在很多神经网络里，它比 sigmoid 更方便居中、梯度更“对称”。

在神经网络中：

- 隐藏层激活函数常选 (\tanh) 或 ReLU 等；
- 可以把每一个隐藏单元看作一种“基函数”，它对输入的响应就是 (\phi_j(x) = \tanh(a_j x + b_j))，网络就是这些基函数的线性组合。

# Stochastic Gradient Descent(SGD)

**核心思想:**
随机抽取部分样本，近似当前梯度

==**实际应用中的折中：小批量梯度下降 (Mini-batch GD)** 在实践中，我们通常不用纯粹的SGD（1个样本），也不用Batch GD（全部样本）。我们采用一个折中方案：每次随机取一小批数据（比如32、64或128个样本），用这个“mini-batch”来计算梯度并更新参数。这既利用了GPU并行计算的优势，又保持了更新的快速和随机性。现在人们通常说的SGD，大多指的都是这种Mini-batch SGD==
$$𝑊 = 𝑊 − 𝜂 \frac{\partial{l}}{\partial{W}}$$
$$b = b − 𝜂 \frac{\partial{l}}{\partial{b}}$$
$$\theta_{t+1} = \theta_t - \mu_t\nabla_{\theta}J(\theta_t)$$
- $𝜂$: learning rate
- $l$: **单个小批量 (mini-batch) 数据** 的损失
- $\nabla_{\theta}J(\theta_t):$ 梯度

**优点:**
这样每次更新都 **非常快**，只需随机抽取32,64,128,256...个样本

**缺点:**
更新方向比较“抖动”，不如全量梯度那么稳定

# Regularization Function

Ridge，Lasso Regularisation都只能用在线性模型

**举例：**
假设我们做一个房价预测，有 3 个特征：
1. $x_1$：房子面积（重要特征）
2. $x_2$：房龄（中等重要）
3. $x_3$：窗帘颜色编码（基本没用）

我们先拟合出一个**未正则化**的线性回归，参数可能是：

$w_1=300,w_2=−20,w_3=5$

你怀疑这是**过拟合**，尤其是没用的 $x_3$ 也被模型利用了，于是考虑加正则化
### Ridge
$$
\begin{align}
&\text{Loss}_{\text{Ridge}} = \sum_{i=1}^n (y_i - \hat{y}_i)^2 +\lambda \sum_{j=1}^d w_j^2\\
&\text{penalty:} \; \lambda \sum w_j^2\\
\end{align}
$$
### 更适合用 Ridge 的典型情况

- 特征很多且高度相关（**多重共线性**），但你觉得：
    - 大部分特征都有点用，不希望完全删掉谁；
- 更在意“稳定性”和“数值平滑”，比如：
    - 模型在不同的数据拆分上参数不要大起大落；
- 常见于：
    - 岭回归（Ridge Regression）、带 L2 的线性/逻辑回归、神经网络的 weight decay 等。

### Lasso
$$
\begin{align}
&\text{Loss}_{\text{Lasso}} = \sum_{i=1}^n (y_i - \hat{y}_i)^2 + \lambda \sum_{j=1}^d |w_j|\\
&\text{penalty:}\;\lambda \sum |w_j|\\
\end{align}
$$
### 更适合用 Lasso 的典型情况

- 你怀疑特征很多，其中只有少数是关键的：
    - 想让模型自动做**特征选择**；
    - 希望最后的模型更“**稀疏，可解释**”：只保留若干非零权重。
- 在高维（特征数量远大于样本数）时尤其常用：
    - 比如基因数据分析、文本特征（成千上万的词）等。
