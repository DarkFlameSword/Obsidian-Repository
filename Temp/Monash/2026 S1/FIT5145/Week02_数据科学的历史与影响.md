5. **数据整理**：数据清洗、转换和整理的过程
6. **整洁数据**：符合特定结构标准的数据格式，便于分析

### 数据科学的历史演变
- **早期先驱**：
  - 数据分析（John Tukey，1962年）
  - 专家系统（1980年代）
  - 机器学习（1980年代）
  - 数据挖掘（1990年代）
- **数据科学的诞生**（2000年左右）：
  - 互联网泡沫（Dot.Com boom）推动数据工程成熟
  - William Cleveland于2001年发表《Data Science: An Action Plan for... the field of Statistics》
  - 数据成为宝贵资产，商业决策压力增大
  - 数据管理和信息社会的发展
  - 互联网公司展示了数据驱动的价值

### 数据科学的影响领域
1. **个人生活**：个性化推荐、健康监测、智能家居
2. **科学**：加速研究进程（如基因组学、天文学）
3. **社会**：城市规划、交通优化、公共政策制定
4. **公益**：灾难响应、公共卫生监测、环境保护
5. **未来**：人工智能伦理、数据隐私、算法公平性

### R和R Markdown简介
- **R语言**：专门为统计计算和图形设计的数据科学编程语言
- **R Studio**：R的集成开发环境（IDE）
- **R Markdown**：将代码、结果和文本结合的可重复报告格式
  - 支持生成HTML、PDF、Word等多种输出格式
  - 便于分享分析过程和结果

### 数据可视化与美学
- 可视化原则：清晰、准确、有效传达信息
- 美学要素：颜色、形状、尺寸、标签的合理使用
- 常用R可视化包：ggplot2、plotly、shiny

### 数据整理
- **定义**：将原始数据转换为适合分析的结构化格式的过程
- **常见任务**：
  - 缺失值处理
  - 数据类型转换
  - 变量重命名
  - 数据筛选和排序
  - 数据合并和连接

### 整洁数据
- **整洁数据原则**（Hadley Wickham提出）：
  1. 每个变量构成一列
  2. 每个观察构成一行
  3. 每种观测单位构成一个表格
- **优点**：
  - 便于使用tidyverse工具集进行处理
  - 提高代码可读性和可重复性
  - 简化数据分析和可视化流程

## Summary
第二周讲座深入探讨了数据科学的历史演变和广泛影响。数据科学起源于多个相关领域，在2000年左右因互联网发展和数据价值认知而成为独立学科。它对个人、科学、社会、公益和未来都有深远影响。讲座还介绍了R语言和R Markdown的基础知识，强调了数据整理和整洁数据的重要性，为后续的数据分析实践打下基础。

## Key Terms

| Term | Definition |
|------|-----------|
| 数据科学历史 | 数据科学从数据分析、机器学习等先驱领域演变而来的过程 |
| R语言 | 一种专门用于统计计算和图形的开源编程语言 |
| R Markdown | 一种将R代码、输出和文本结合的可重复报告格式 |
| 数据整理 | 将原始数据清洗、转换和重组为适合分析格式的过程 |
| 整洁数据 | 符合"每个变量一列、每个观察一行"原则的数据组织形式 |
| 数据可视化 | 使用图形表示数据，以便更直观地理解模式和关系 |
