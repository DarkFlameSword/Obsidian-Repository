# Tidy data

**Definition:**
is an approach to structuring how you store and use data
- 标准化你的数据可以让你对所有数据进行标准化
- 减少持续处理数据的需求

Messy data:
- 由于数据和列的数值和名称模糊，难以理解
- 数据没有很好地组织
