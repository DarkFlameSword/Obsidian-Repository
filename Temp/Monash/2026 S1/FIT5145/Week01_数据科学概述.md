## Summary
第一周讲座介绍了 FIT5145 数据科学基础课程的整体框架。课程由 Clayton 和马来西亚校区的多位讲师共同授课，涵盖了数据科学的概述、历史、商业模式、数据分析方法、数据管理及未来趋势。学生需要具备基本的数学、编程和批判性思维能力，课程评估主要基于 R 和 Shell 脚本的编码任务。学习资源丰富，但需要合理规划时间，聚焦重点内容。

## Key Terms

| Term | Definition |
|------|-----------|
| 数据科学 | 一门跨学科领域，结合统计学、计算机科学和领域知识，从数据中提取见解并支持决策 |
| 大数据 | 数据量巨大、复杂度高，传统数据处理工具难以处理的数据集 |
| 监督数据分析 | 使用带有标签的训练数据来构建预测模型的分析方法 |
| 无监督数据分析 | 在没有标签的数据中发现模式和结构的分析方法 |
| R 和 R Studio | 数据科学中常用的编程语言和集成开发环境 |
| Shell 脚本 | 在命令行环境中自动化任务的脚本 |
